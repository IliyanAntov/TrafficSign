Admin app -> Server

"GET ":
    1)  "devices"
        {response}: 
            [message 0]: <Number of available devices>
            [message 1-n]: <Device IMEI>
    2)  "details <IMEI>"
        {response}:
            [message 0]: <Device details string>

"SET <Target device IMEI> ":
    1)  "speed <Speed limit>" 
        {response}:
            [message 0]: <Response message>
    2)  "warning <Warning sign>"
        {response}:
            [message 0]: <Response message>


Server -> uC

"SET ":
    1)  "spd <Speed limit>" = speed limit traffic sign (Red circle)
    2)  "wrn <Warning sign>" = warning traffic sign
            -"stp" = Stop sign
            -"gnr" = General warning (exclamation mark)
            -"tfl" = Traffic light
            -"nen" = No entry
            -"fon" = Forward only
            -"lon" = Left only
            -"ron" = Right only

"GET ":
    1)  "dtl" = device details
